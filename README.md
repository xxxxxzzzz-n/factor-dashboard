# Дашборд для автоматического факторного анализа падения продуктовых метрик

## О проекте

Проект разработан в рамках прикладной проектной практики магистерской программы «Бизнес-аналитика и системы больших данных» НИУ ВШЭ. Цель проекта — разработка системы автоматического факторного анализа изменений продуктовых метрик с визуализацией результатов в виде интерактивного дашборда. Система включает формализованный метод аддитивной декомпозиции изменений метрик по сегментам, веб-дашборд для интерактивного анализа результатов и механизм автоматической интерпретации (локальный + LLM-ready архитектура).

## Содержимое репозитория

- `data_raw/` — синтетические данные и агрегированная витрина;
- `factor_analysis.yaml` — конфигурация и логика факторного анализа;
- файлы веб-дашборда (HTML, CSS, JavaScript);
- `data/` — итоговые JSON-результаты факторного анализа;
- `docs/` — отчетные материалы проекта.

## Запуск дашборда

Дашборд реализован как статический веб-сайт и доступен по ссылке: `https://xxxxxzzzz-n.github.io/factor-dashboard/`

Python, SQL, JavaScript, HTML/CSS, JSON, GitHub Pages.
